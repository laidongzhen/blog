# 写博客文章注意点

1.标点符号尽量使用英文符号

2.不用直接写'watch','computed'等选项，需要加个英文引号，要不会报错

3.引用的图片等资源一定是需要存在在本地的，要不会报错

4.在typora中，图像设置格式：

复制到指定路径

./${filename}
